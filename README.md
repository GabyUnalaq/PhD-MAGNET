# Multi-Agent Graph Network for Decentralized Traffic Navigation

## Setup

I am using [uv](https://docs.astral.sh/uv/pip/environments/) to set my python env. The `requirements.txt` file contains the libraries needed to execute this project.

I am using `python 3.10` and `torch` with cuda support for cuda 12.1. For the Graph specific torch library, I had to install also `torch-geometric`.
